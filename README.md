# KALKI System Core

Wise, Powerful, and Righteous AI Interface for Work Management.

## Quick Start

1. Install: `pip install fastapi uvicorn google-genai aiosqlite`
2. Run Backend: `python3 -m uvicorn main:app --host 0.0.0.0 --port 8000`
3. View HUD: `http://localhost:8000`
4. Run CLI: `python3 kalki_cli.py` (requires `GOOGLE_API_KEY` for full cognitive functions)
